Overflow menu item is used as the main slot's value for the <code class="text-green">BCMSOverflowMenu</code>.

Example:

```tsx
<BCMSOverflowMenu>
  <BCMSOverflowMenuItem
    text="Edit"
    icon="edit"
    description="Edit description"
    onClick={() => {}}
  />
  <BCMSOverflowMenuItem text="Notifications" icon="bell" onClick={() => {}} />
  <BCMSOverflowMenuItem
    text="Remove"
    icon="trash"
    theme="danger"
    onClick={() => {}}
  />
</BCMSOverflowMenu>
```
